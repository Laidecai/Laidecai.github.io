# xianzun 项目目录结构说明

```
xianzun/
├── index.html          ← 主页面入口
│
├── assets/             ← 原始资源（已存在）
│   └── fangyuan.png    ← 方源人物立绘
│
├── images/             ← 图片素材（背景、图标、纹理等）
│
├── css/                ← 样式表文件
│
├── js/                 ← JavaScript 脚本文件
│
├── audio/              ← 音频文件（背景音乐、音效）
│
├── fonts/              ← 本地字体文件（.woff2 / .ttf）
│
├── videos/             ← 视频素材（背景视频、动画）
│
├── components/         ← 可复用 HTML 组件片段
│
└── docs/               ← 项目文档
    └── 目录结构说明.md  ← 本文件
```

---

## 各目录与 index.html 的映射关系

### 1. `assets/` — 原始图片资源

当前引用：
```html
<img src="./assets/fangyuan.png" alt="方源" />
```

**用途**：存放人物立绘、角色原画等核心视觉素材。

---

### 2. `images/` — 图片素材

**引用方式**：`./images/xxx.png`

**适合存放**：
| 文件类型 | 示例 |
|---------|------|
| 背景纹理 | `bg-stars.webp`, `bg-nebula.jpg` |
| UI 图标 | `icon-github.svg`, `icon-arrow.png` |
| 装饰元素 | `cicada-wing.png`, `rune-glow.png` |
| 渐变/遮罩 | `mask-character.png` |

**在 index 中引用示例**：
```html
<!-- 背景图片 -->
<div style="background: url('./images/bg-stars.webp')"></div>

<!-- 图标 -->
<img src="./images/icon-github.svg" alt="GitHub" />
```

---

### 3. `css/` — 样式表文件

**引用方式**：`<link rel="stylesheet" href="./css/xxx.css">`

**适合存放**：
| 文件 | 用途 |
|-----|------|
| `style.css` | 主样式表（可把 index.html 中的 `<style>` 移入） |
| `animations.css` | 动画关键帧 |
| `responsive.css` | 响应式布局 |
| `themes.css` | 主题变量 |

**在 index 中引用示例**：
```html
<head>
  <link rel="stylesheet" href="./css/style.css">
  <link rel="stylesheet" href="./css/animations.css">
</head>
```

---

### 4. `js/` — JavaScript 脚本文件

**引用方式**：`<script src="./js/xxx.js"></script>`

**适合存放**：
| 文件 | 用途 |
|-----|------|
| `galaxy.js` | Three.js 银河粒子系统 |
| `interactions.js` | 鼠标水波纹、玻璃扫光等交互 |
| `glitch.js` | 人物故障/能量效果 |
| `scan-effects.js` | 扫描框 & 字符效果 |
| `navigation.js` | 翻页导航、滚动控制 |

**在 index 中引用示例**（放在 `</body>` 前）：
```html
<script src="./js/galaxy.js"></script>
<script src="./js/interactions.js"></script>
<script src="./js/glitch.js"></script>
```

---

### 5. `audio/` — 音频文件

**引用方式**：
```html
<audio src="./audio/xxx.mp3"></audio>
```

**适合存放**：
| 文件 | 用途 |
|-----|------|
| `ambient.mp3` | 背景氛围音（古琴、风声） |
| `click.wav` | 点击音效 |
| `hover.wav` | 悬停提示音 |
| `scroll.mp3` | 翻页翻书声 |

**在 index 中引用示例**：
```html
<audio id="ambient-audio" loop src="./audio/ambient.mp3"></audio>
```

---

### 6. `fonts/` — 本地字体文件

**引用方式**（CSS 中 `@font-face`）：
```css
@font-face {
  font-family: 'MyFont';
  src: url('./fonts/myfont.woff2') format('woff2');
}
```

**用途**：如果不想依赖 Google Fonts CDN，可将字体下载到本地引用。

**在 index 中引用示例**：
```css
@font-face {
  font-family: 'ZCOOL QingKe HuangYou';
  src: url('./fonts/ZCOOLQingKeHuangYou-Regular.woff2') format('woff2');
}
```

---

### 7. `videos/` — 视频素材

**引用方式**：
```html
<video src="./videos/xxx.mp4" autoplay loop muted></video>
```

**适合存放**：
| 文件 | 用途 |
|-----|------|
| `bg-galaxy.mp4` | 银河背景视频 |
| `cicada-anim.mp4` | 春秋蝉动画 |
| `particle-overlay.webm` | 粒子叠加层 |

---

### 8. `components/` — HTML 组件片段

**用途**：存放可复用的 HTML 片段，方便后期通过 JS 动态加载。

**示例**：
| 文件 | 内容 |
|-----|------|
| `header.html` | 页头导航 |
| `footer.html` | 页脚 |
| `hero-section.html` | 英雄区代码 |
| `character-card.html` | 身份卡组件 |

**在 index 中引用示例**（配合 JS 加载）：
```javascript
fetch('./components/footer.html')
  .then(r => r.text())
  .then(html => document.getElementById('footer-placeholder').innerHTML = html);
```

---

## 最佳实践建议

1. **当前阶段**：所有样式和 JS 都在 `index.html` 中内联，开发方便
2. **后期拆分**：当文件变大时，将 `<style>` 内容移到 `css/style.css`，将 `<script>` 按功能拆到 `js/` 下各文件
3. **引用路径**：所有路径均使用相对于 `index.html` 的 `./xxx/` 相对路径
4. **新增资源**：根据文件类型放入对应目录，在 `index.html` 中用相对路径引用即可
